This is the first transmission from the rebuilt site. The old portfolio served me well, but it was hand-written HTML I dreaded touching — so I rebuilt it on **Astro**, moved the whole thing to **Codeberg**, and gave myself a place I'll actually enjoy posting to.

This is where I'll write about the things I spend my days on: typed Python, data platforms, monorepo tooling, developer experience, and the occasional war story from the energy grid. Some posts will live here; others I'll link out to from wherever they're published.

To add a new article, I drop a Markdown file into `src/content/articles/`. To link an external post, I set its `externalUrl` and it shows up in the list pointing outward. That's the whole workflow — which is exactly the point.

More soon.
